Template Driven Form

Advantage:
  • Less coding is required in the component class.
  • Easy to understand and stat with.
Disadvantage:
  • Difficult to add controls dynamically at runtime.
  • unit testing challenging with TD forms.

Reactive Form

Advantage:
  • Structure of the forms is defined in typeScript class.
  • Creating dynamic controls is easier.
  • Easy to unit test.
Disadvantage:
  • Most of things are done by writing code.

What is form array?

A FromArray is a way to manage a collection of form control in angular. The form control can be a formGroup,FormControl, or another FormArray

In Angular we can group form control two ways
  • Using FormGroup
  • Using FormArray
Difference between
  • FormGroup stores the form controls in the form of key values pair in an object
  • FormGroup stores the form controls as an element of an array
